The parameters requrie users to first enter the file path of the image, then enter in the block-size used to create the histograms. After the user has entered a block size, which must be 2^n, the vectors will populate on top of the image. The resulting magnitude and direction of each vector create the final output

Mini Search Engine

This project was written in the C-programming language. The purpose of this program is to search through a set of documents to determine the relevance of the documents with respect to a search phrase/query. This was implemented using hash tables and linkedlists to organize the contents of each document. Using the term frequency-inverse document frequency method, a list of scores is generated to determine the most to least relevant documents.
